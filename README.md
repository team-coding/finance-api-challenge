### Desafio IGTI 

#### Como executar a api

- `Abra o projeto no container dev (Remote in container)`

- `Configure o seu arquivo .env (Utilize o .env.example como modelo)`

- `$ yarn install:all`

- `$ yarn build`

- `No seu terminal fora do container execute o comando: `

- `$ yarn start:backend`
